== Description ==

Allows you to quickly create a maintenance/coming-soon page. You can use this plugin whenever your site is down for maintenance, currently going any upgrade or is in development. Built in options panel to add logo, background image, social icons, color, subscribe field and many more.

Get free support at http://themegrill.com/support-forum/

check the demo at http://themegrill.com/plugins/maintenance-page/

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Maintenance Page plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the Settings->Maintenance Page in your dashboard.
